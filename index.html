Mathematical problems

Scope

Today, we only consider the following problem variant:
  • The surface area is already determined
  • The number of robots has been decided on
  • All parcels to be sorted are known in advance

The problems

  • Allocation: which robots brings which parcel?
    • Pick up parcel close by
    • Fast route is available
    • Minimize total completion time
  • Path finding: which route should the robot take?
    • Short route
    • Avoid collisions with other robots
    • Minimize total completion time

Solution techniques

We will consider two.

Technique 1: Fast, simple decision rule

  • Fast, simple
  • Solutions often not that good
  • Often, scale very well
  • Also called heuristic

Technique 2: Solving the problem completely

  • Solutions optimal
  • For hard problems:
    • Slow, more complicated
    • Don't scale

We can do this using linear programs

Linear programming

A generic modeling technique, with three steps:

  1. Define variables

    $x^s_{r, i}:$ whether robot $r$ delivers parcel $i$ first

  2. Define restrictions on those variables

    $x^s_{r, i} = 1$ if robot $r$ delivers parcel $i$ first, $x^s_{r, i} = 0$ otherwise

    One request is first: $x^s_{r, 1} + x^s_{r, 2} + \ldots + x^s_{r, n} = 1$

  3. Define the objective function

    $\max_r c^s_{r, 1} x^s_{r, 1} + c^s_{r, 2} x^s_{r, 2} + \ldots + c^s_{r, n} x^s_{r, n}$

Then, solve the formulated problem using a numerical solver
